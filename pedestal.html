Right Control Display Unit (CDU)

P/N[Enter P/N Later]
First Officer's text interface console syncing information directly into the FMC logic cross-talk loop.

Speed Brake & Pitch Trim Mechanical Quadrant

P/N[Enter P/N Later]
Houses the mechanical lever for Speed Brakes / Spoilers (Down, Armed, Flight Detent, Up parameters) along with the Alternate Pitch Trim switches and parking brake handle.

Thrust Levers (Throttle Assembly)

P/N[Enter P/N Later]
Dual electronic thrust levers linked to the A/T (Autothrottle) servo systems. Features integrated A/T Disconnect and TO/GA (Takeoff/Go-Around) click toggles.

Flap Selection Handle

P/N[Enter P/N Later]
Controls wing high-lift configuration trailing flaps and leading edge slats. Detents range through gates 1, 5, 15, 20, 25, to 30. Includes an Alternate Flap control panel armor setup.

Fuel Control Switches

P/N[Enter P/N Later]
Two-position toggles (RUN / CUTOFF) controlling high-pressure fuel valves to command starting sequence or emergency fire isolation fuel starvation metrics.

Radio Tuning & Audio Select Panel (Left)

P/N[Enter P/N Later]
Primary VHF/HF tuning array paired with transmission selection key indicators for microphones, interphones, and speaker gain pots.

Secondary Communication & Navigation Monitoring Module

P/N[Enter P/N Later]
Lower left console panel providing radio link fallbacks and receiver markers for monitoring Morse code idents over VOR, ADF, or ILS stations.

Flight Deck Access Door Security Panel

P/N[Enter P/N Later]
Toggle interface managing cabin request entries, featuring UNLKD, AUTO, and DENY override selection lock configurations.

Engine & APU Fire Bottle Manual Squib Discharge

P/N[Enter P/N Later]
Guarded backup activation push buttons targeting armed fire extinguisher squib canisters inside active structural bays.

Center Maintenance Control Display Unit

P/N[Enter P/N Later]
Third shared multi-purpose interface console. Commonly dedicated to SATCOM management, ACARS telemetry pipelines, or onboard troubleshooting diagnostic indices.

Aileron & Rudder Trim Module

Contains mechanical trim input pots including the proportional rotational knob modifying zero-force aerodynamic alignment parameters on the rudder surface loop.

Integrated Cockpit Data Link Printer

Thermal printing apparatus printing weather briefs, oceanic text clearances, or structural operational parameters routed via ACARS datalink modules.

Radio Tuning & Audio Select Panel (Right)

P/N[Enter P/N Later]
First Officer's dedicated primary radio and cabin communications terminal selector board interface.

ATC Transponder & TCAS Selection Panel

P/N[Enter P/N Later]

Evacuation Alert & Console Lighting Trim
