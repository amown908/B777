EFIS Control Panel — Captain

P/N-
Manages map presentation format scales, weather radar profiling overlays, and barometric altimeter baseline targets.

Mode Control Panel (MCP)

P/N-
Central automation command deck housing autopilot, autothrust, and flight director mode configurations.

EFIS Control Panel — First Officer

P/N-
Independent display formatter managing symbology, visual filters, and minimum limits logic for the F/O.

Primary Flight Display — Captain

P/N-
Displays core parameters including airspeed velocity columns, attitude horizons, altitude tapes, and vertical speed flags.

Navigation Display — Captain

P/N-
Presents active flight route planning, geographic map updates, terminal waypoint fixes, and weather hazard radar projections.

Brake Accumulator Pressure Indicator

P/N-
Monitors standby emergency hydraulic lines designated for main landing gear wheel braking loops.

Inboard Display Selector — Captain

P/N-
Auxiliary system routing selector mapping primary screen display formats on backup visual tubes.

Integrated Standby Flight Display (ISFD)

P/N-
Isolated backup instrument calculating situational orientation using decoupled auxiliary sensor inputs.

Upper EICAS

P/N-
Hosts primary engine operation parameters, gross fuel monitoring lines, and real-time crew alert system status logs.

Landing Gear Control Panel

P/N-
Houses structural landing gear sequencing handle, manual lock override nodes, and autobrake profiles.

Lower EICAS

P/N-
Dedicated layout display monitoring secondary mechanical structures, configurations, and component checklists.

Captain FMC (Control Display Unit)

P/N-
Captain's primary programming console input node communicating route metrics back into the Flight Management Computer.

F/O FMC (Control Display Unit)

P/N-
First Officer's key interface entry matrix tracking flight paths, leg metrics, and navigational logistics.

Navigation Display — First Officer

P/N-
Presents independent terrain mappings, terminal routing markers, and tracking structures for the F/O's instrumentation stack.

Primary Flight Display — First Officer

P/N-
Tracks independent airspeed scales, pressure parameters, and orientation vectors for safety cross-checks.

Inboard Display Selector — F/O

P/N-
First officer's diagnostic selector managing localized format overrides.

FMC Selector Switch

P/N-
Determines which Flight Management Computer holds master cross-talk command authority over active lines.
